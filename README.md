Jio Hotstar Streaming Platform Dashboard (Power-BI)
This project analyzes streaming platform data from Jio Hotstar using Power BI to generate insights on content distribution and viewer trends.

Tools Used
• Power BI

• Data Modeling

• DAX

• Data Visualization

# Dashboard Features
• Total Movies and TV Shows

• Genre distribution analysis

• Release year trends

• Content type comparison

• Country-wise content distribution

# Key Insights
• Growth of streaming content over time

• Popular genres on the platform

• Distribution of movies vs TV shows

• Content availability by country

# Files
• jio hotstar dashboard.pbix – Power BI dashboard

• jio_hotstar_dataset.csv – Dataset used for analysis
